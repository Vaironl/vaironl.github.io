Social Blood

Django Web Application.

This Django web application was a prototype used to demonstrate the ability of having a website were doctors in countries with the lack of resources could find blood donators in case of catastrophes. The application helped me to understand how to develop a prototype web application in Django from scratch. It also introduced me to the complexity of building a web application which had the ability to sort through data and used third party websites (Facebook in this case) as a form of account registration and login option. Please note the application is now defunct, however, the source code is still available.

  • Date: December 2016
  • Client: Social Blood
  • Category: Web Application
